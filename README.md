# Stock Market Backtesting & Prediction App

##Overview
This project is a stock market analysis tool that allows users to:
1.Backtest trading strategies using historical data.
2.Predict future stock prices using Machine Learning.
3.Analyze performance using financial metrics.
4.Interact through a simple Streamlit web application.

##Features
- Backtesting: SMA, RSI, MACD, Mean Reversion  
- ML Models: Linear Regression, LSTM, RNN, Random Forest  
- 5-day Stock Price Prediction  
- Performance Metrics: Sharpe Ratio, Drawdown, Returns  
- Interactive Streamlit Dashboard  

##Tech Stack
Python, Pandas, NumPy, yFinance, Scikit-learn, TensorFlow, Streamlit, Plotly
